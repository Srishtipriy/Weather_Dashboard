# Weather_Dashboard
A weather Dashboard with smooth animations accurate results with other interactive weather metrics like humidity ..etc

AI-Powered Weather Intelligence Dashboard (Web Application)
• Developed a responsive weather intelligence dashboard that retrieves real-time weather data using REST APIs and displays current conditions, hourly forecasts, and 5-day forecasts.
• Implemented dynamic UI with weather-based gradient backgrounds and animated environmental effects (wind-driven leaves, floating flowers, and atmospheric transitions).
• Built an AI-powered weather advisor chatbot enabling users to ask contextual questions such as travel suitability, drying clothes outdoors, or planning activities based on weather conditions.
• Integrated multiple weather metrics including temperature, humidity, wind speed, air quality index, visibility, and sunrise/sunset schedules with interactive visualization cards.
• Designed a modern glass-morphism interface with responsive layouts, smooth animations, and optimized user interactions for enhanced UX.

Technologies: HTML, CSS, JavaScript, OpenWeatherMap API, REST APIs.
